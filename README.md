# ChatPortal

A full-featured SaaS AI Chat Platform built with ASP.NET Core MVC and C#.

## Features
- JWT Authentication (Login, Register, 2FA)
- AI Chat with streaming responses (OpenAI-compatible)
- Dashboard with analytics, reports, and insights
- Team collaboration and messaging
- Billing and subscription management
- REST API with JWT Bearer auth
- Real-time notifications via SignalR

## Tech Stack
- ASP.NET Core 8.0 MVC
- Entity Framework Core (SQL Server)
- SignalR for real-time features
- JWT Authentication
- OpenAI-compatible AI integration

## Getting Started
1. Clone the repository
2. Update appsettings.json with your connection string and API keys
3. Run dotnet ef database update
4. Run dotnet run
